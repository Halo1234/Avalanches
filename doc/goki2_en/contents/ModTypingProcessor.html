ModTypingProcessor tags

Reference of tags available for ModTypingProcessor.

Word data for the typing game can be entered in Excel.
Edit /src/data/typing_words/words.xls.
Specify the string to be displayed as a caption in the caption column.
Specify the character string to be entered in the ruby column in katakana.
Run /tools/make_word/run.bat to create the data.
Word data will be generated in the same location, so please copy it to the project folder and load it with the [load_typing_words] tag.

load_roman_table(No alias)

Typing operations
aliasattributeRequired?valueDescription
NonelanguageyesStorage nameSpecifies the Romanization table for a specific language.
The storage name specified here specifies the Roman table generated by the tool "make_roman_table".
Do not specify an extension.
To create a Roman alphabet table, first edit /src/data/roman_alphabet/japanese.xls.
You can understand how to edit by looking at the content, but a simple explanation is available in /src/data/roman_alphabet/readme.txt.
After editing, when you run /tools/make_roman_table/japanese.bat, "japanese.dic" and "japanese_symbol.dic" are generated under /tools/make_roman_table/.
Place these two files in the development environment and read them using this tag.
Example:
; Load "japanese.dic" and "japanese_symbol.dic".
[load_roman_table language=japanese]

load_typing_words(No alias)

Typing operations
aliasattributeRequired?valueDescription
bufstorageyesStorage nameSpecify the storage name for the typing word.
Read typing words.
To create typing words, edit the word data by referring to /stc/data/typing_words/words.xls.

Next, run the *.bat file, copy the generated *.dic file to the development environment, and use this tag to read the typing word.

show_typing_target(No alias)

Typing operations
aliasattributeRequired?valueDescription
NonecaptionnoCaption stringSpecifies the caption string of the typing object to be displayed.
If you specify "random", a word will be read at random from the typing words that have been read.
Default is "random".
NonerubynoRuby stringSpecifies the ruby string of the typing object to be displayed.
If you specify "random", a word will be read at random from the typing words that have been read.
Default is "random".
NonestorageyesStorage nameSpecify the image storage name for the typing object.
If omitted, it will be loaded randomly from the image storage name specified in [typing_config].
Nonepositionnorandom or centerSpecify the display position.
If you specify random/center here, both left and top will be treated as random/center.
Noneleftnoin pixels or random or centerSpecifies the left corner position.
If you specify random, it will be displayed at a random position.
If you specify center, it will be displayed in the center.
Nonetopnoin pixels or radom or centerSpecifies the top corner position.
If you specify random, it will be displayed at a random position.
If you specify center, it will be displayed in the center.
Displays typing objects.
Return to genre/tag list
