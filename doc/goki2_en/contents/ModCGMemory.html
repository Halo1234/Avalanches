ModCGMemory tags

Reference of tags available in ModCGMemory.

CG/reminiscence data will be entered in Excel.
Edit /src/data/cgmemory/list.xlsx.
Specify the button image to be displayed as a thumbnail in the thumbnail column.
In the CG column, you can specify any number of CGs to be displayed (listed without extensions) by separating them with line breaks.
In the recall column, specify the storage file name (written without extension) to be executed as a recall.
Run /tools/make_cgmem/run.bat to create data.
The data will be generated in the same location, so copy it to the project folder and load it with the [load_cgmemory] tag.

For CG, the image loaded with the load_cg tag will be opened.

open_memory(No alias)

CG and memory.
Open reminiscence.
After opening, execute the end_memory tag and that will be the end of the recollection.
The recollection will be played from the label position immediately before this tag is executed.
Return to genre/tag list

end_memory(No alias)

CG and memory.
Close reminiscence.
After passing this tag, the recollection will be opened.
Also, if reminiscence is being played, after this tag is executed, the screen will return to the reminiscence screen.
Return to genre/tag list
