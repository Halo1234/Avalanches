nonevagueno(0 or more)Specifies the vague area value to apply to the universal transition.
nonefromno"left" or "top" or "right" or "bottom"Specify the direction from which the back page image appears in the scroll transition.
nonestayno"stayfore" or "stayback" or "nostay"(Default)In a scroll transition, either the image on the front page stays in place and the image on the back page scrolls in from the outside ("stayfore"),
or the image on the front page scrolls out from the back, and the image on the back page scrolls out.
Specifies whether the image on the page remains in place ("stayback"), or whether the images on the front and back pages are linked and moved together ("nostay"). Specify.

move(No alias)

Layer operations
aliasattributeRequired?valueDescription
nonelayernoStringSpecify the name of the image layer.
It can be between 0 and N. (N is the number of standing image layers specified by cv_layers - 1)
If omitted, base is assumed to be specified.
nonepagenofore or backPage name of the target layer.
If you specify fore, the layer on the fore page will be targeted.
If you specify back, the layer on the back page will be targeted.
If omitted, it is assumed that fore is specified.
nonesplinenotrue or falseIf you specify true, the layer will move on a line that is interpolated using a B-spline curve at the position specified by the path attribute.
If false is specified, the layer will move along a line that is linearly interpolated from the position specified in the path attribute.
If you specify true, please specify at least 2 points for path.
nonetimenomillisecondsSpecifies the time required for the layer to move a point from the point specified by path.
Therefore, the movement takes time equal to the number of intervals between points traversed * the value of this attribute.
However, if you specify a value other than 0 for the accel attribute, the time it takes to pass one point will change (the time required for the entire movement is the number of sections passed * the value of this attribute).
nonedelaynomillisecondsSpecifies the time in ms before automatic layer movement begins.
If omitted, it is assumed to be 0.
nonepathno"(x1,y1,opacity1) (x2,y2,opacity2) ..."Specify the movement position.
Specify the movement position by separating x (left edge position of the layer), y (top edge position of the layer), and opacity (layer density) with commas, spaces, or parentheses ( ).
Layers are moved between these using spline or linear interpolation.
opacity specifies the layer's display density. The density of the layer also changes continuously as you move from point to point.
If you only want to change the display density without moving the layers, specify the same position for all layers.
noneaccelnoA number less than -1 or 0(Default) or greater than 1Specify whether to perform accelerated movement.
If you specify 0, the movement process will proceed at a constant rate from beginning to end (each passing point will be passed at equal intervals of time). If omitted, 0 is assumed.
A number less than -1 will start faster and gradually become slower.
A number greater than 1 will start slower and gradually speed up.
You can also specify a real number.
Move the layer.
Return to genre/tag list

noneindexnoIntegerDo not specify it for the background layer.
Specify the stacking order. The higher the value, the closer it will be displayed.
noneopacityno0-255Specifies the opacity of the layer specified by the layer attribute.
Defaults to 255 (totally opaque). In other words, if you load an image without specifying this attribute, even if the layer's opacity is not 255, it will be reset to 255.
If you specify 128, the transparency will be about half, and it will be visible.
You can also make a layer invisible by setting opacity=0, but it doesn't make it completely transparent; it doesn't make it invisible.
Setting this attribute does not change the layer image itself. What changes is the display state.
