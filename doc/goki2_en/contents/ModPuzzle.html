ModPuzzleのタグ

A reference to tags available in ModPuzzle.
Call the following in the puzzle piece animation file.

Call the following at the end of the landing animation.

[call_handler message=onLanding]

Call the following at the end of the erasing animation.

[call_handler message=onErase]

Several events are defined in the puzzle piece animation script.
Labelexplanation
*waitingImplements a waiting animation.
*landingImplements a landing animation.
*eraseImplements an erasing animation.
