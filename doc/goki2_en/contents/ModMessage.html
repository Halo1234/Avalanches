ModMessage tags

A reference of tags available in ModMessage.

ch_mod_message/ruby_mod_message/p_mod_message/l_mod_message/r_mod_message/erase_message/clear_message_layers_with_history/reset_message_layers_with_history, which is the actual state of ch tags, etc., has been implemented.
using_mod_message tag will redirect ch/ruby/p/l/r/er/cm/ct tags to these tags respectively.

Unlike KAG3, the message layer can save the contents, so there is no need to insert [cm] tags between each saveable label.
