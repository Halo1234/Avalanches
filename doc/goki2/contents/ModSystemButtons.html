system_button_option(別名なし)

効果音操作
別名属性必須?値説明
なしvisiblenoストレージ名セーブ画面の背景画像を指定します。
なしbufferno効果音バッファ番号効果音を再生するバッファ番号を指定します。
なしclick_soundnoストレージ名クリック時に再生する音声ファイル名を指定します。
なしclick_soundnoストレージ名クリック時に再生する音声ファイル名を指定します。
なしenter_soundnoストレージ名マウス侵入時に再生する音声ファイル名を指定します。
なしleave_soundnoストレージ名マウスが出て行った時に再生する音声ファイル名を指定します。
システムボタンの設定を変更します。

system_button(別名なし)

効果音操作
別名属性必須?値説明
なし<id>_storagenoストレージ名ボタンに使用する画像を指定します。
<id>には以下の文字列が指定できます。
log : ログボタン
system : システム画面呼び出しボタン
save : セーブ画面呼び出しボタン
load : ロード画面呼び出しボタン
skip : スキップ開始ボタン
auto : オートモード開始ボタン
hidden : メッセージ消去ボタン
なし<id>_leftno位置ボタンの表示位置を指定します。
なし<id>_topno位置ボタンの表示位置を指定します。
なし<id>_widthnoサイズボタンの幅を指定します。
なし<id>_heightnoサイズボタンの高さを指定します。
なし<id>_captionno文字列ボタンのキャプションを指定します。
なし<id>_caption_colorno0xRRGGBBボタンのキャプションカラーを指定します。
なし<id>_colorno0xRRGGBBボタンの背景色を指定します。
なし<id>_opacityno0〜255ボタンの不透明度を指定します。
なし<id>_fontnoフォント名ボタンのフォント名を指定します。
なし<id>_font_heightnoフォントサイズボタンのフォントサイズを指定します。
システムボタンの項目を設定します。
