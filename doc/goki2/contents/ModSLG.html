ModSLGのタグ

　ModSLG で使用可能なタグのリファレンスです。

ダメージ計算などは/src/goki2/system/ModSLGCalc.tjsで行っています。
基本的にはファイル先頭にある定義をgame_editorで定義した名前に変えるだけで動きますが
計算式などを変更したい場合はTJS2で記述する必要がありますので吉里吉里２のマニュアルなどを参照してください。

例えばHPパラメータ名を変更したい場合はvar pHPName = "HP"をvar pHPName = "体力"などとします。
/*----------パラメータ名定義 ココから*/
// HPパラメータ名
var pHPName = "HP";
// strパラメータ名
var pStrName = "str";
// intパラメータ名
var pIntName = "int";
// vitパラメータ名
var pVitName = "vit";
// agiパラメータ名
var pAgiName = "agi";
// dexパラメータ名
var pDexName = "dex";
// lukパラメータ名
var pLukName = "luk";
/*----------ココまで*/

slgprepareタグは使用後必ずsタグで停止してください。
またセーブ/ロードを有効化する場合セーブ可能なラベルをsタグの前に記述してください。
例：
@slgprepare start

*force_store|
@s

キャラクタの*.asdファイルには攻撃の瞬間を以下のタグで指定してください。

[call_handler message=onAttack]

同じく回復の瞬間を以下のタグで指定してください。

[call_handler message=onRest]

SLG操作

なしitem_max_countno整数値アイテムの所持限界数を指定します。
なしskill_max_countno整数値スキルの所持限界数を指定します。

なしgame_over_labelnoラベル名ゲームオーバー時に実行するラベル名を指定します。

なしenemy_count_storagenoシナリオストレージ名敵の数がenemy_count以下になった時に実行するシナリオストレージ名を指定します。
なしenemy_count_labelnoラベル名敵の数がenemy_count以下になった時に実行するラベル名を指定します。
なしmove_speedno整数値マップ上の移動速度を指定します。
なしrebirth_characternotrue or falsetrueを指定すると次のステージでキャラクタが復活します。
falseを指定すると死んだキャラクタは復活しません。
なしmoneyno整数値資金を設定します。
なしsound_bufferno整数値システム全体の効果音で使用するバッファ番号を指定します。
なしclick_soundnoストレージ名システム全体のクリック時に再生するサウンドストレージ名を指定します。
なしenter_soundnoストレージ名システム全体のマウスエンター時に再生するサウンドストレージ名を指定します。

[slgconfig cursor_sound_buffer=0 cursor_enter_sound=se002.ogg]
[slgconfig bgm_storage=bgm001.ogg battle_bgm_storage=bgm002.ogg]
