move(別名なし)

レイヤ操作
別名属性必須?値説明
なしtimenoミリ秒単位レイヤが path を指定した点から点を移動するのに必要な時間を指定します。
そのため、移動には、通過する点間の区間の数×この属性の値の時間がかかります。
ただし、accel 属性に 0 以外を指定した場合は 一点を通過する時間は 変化します ( 移動全体に要する時間は 通過する区間の数×この属性の値です )。
なしdelaynoミリ秒単位レイヤの自動移動が開始されるまでの時間を ms で指定します。
省略すると 0 であるとみなされます。
なしpathno"(x1,y1,opacity1) (x2,y2,opacity2) ..."移動位置を指定します。
移動位置は、x(レイヤ左端位置), y(レイヤ上端位置), opacity(レイヤ不透明度) を３つ づつ、カンマや空白、( ) で区切って指定します。
opacity にはレイヤの表示濃度を指定します。レイヤの濃度も点から点を移動する 間に連続的に変化します。
なしaccelno-1 未満の数または 0(デフォルト) または 1 より大きい数加速度的な動きを行わせるかどうかを指定します。
0 を指定すると、最初から最後まで一定の割合で移動処理が進行し ます ( 各通過点を等間隔の時間で通過します )。省略すると 0 が指定され たと見なされます。
-1 未満の数を指定すると、最初は早く、徐々に遅くなります。
1 より大きい数を指定すると、最初は遅く、徐々に早くなります。
実数を指定することもできます。
レイヤを移動させます。
ジャンル・タグ一覧に戻る

layer_move(別名なし)

レイヤ操作
別名属性必須?値説明
なしclearnotrue 又は false移動パスをクリアするかどうかを指定します。
true を指定すると移動パスをクリアします。
false を指定すると何もしません。
デフォルトはfalseです。
なしtransformnotrue 又は false変形（回転拡大縮小）を行うかどうかを指定します。
true を指定すると変形を行います。
false を何もしません。
なしchildrennotrue 又は false変形に子レイヤも巻き込むかどうかを指定します。
true を指定すると子も巻き込みます。
false を指定すると何もしません。
なしloopingnotrue 又は falseループするかどうかを指定します。
true を指定するとループします。
false を指定すると何もしません。
なしtimenoミリ秒単位レイヤが path を指定した点から点を移動するのに必要な時間を指定します。
そのため、移動には、通過する点間の区間の数×この属性の値の時間がかかります。
ただし、accel 属性に 0 以外を指定した場合は 一点を通過する時間は 変化します ( 移動全体に要する時間は 通過する区間の数×この属性の値です )。
なしdelaynoミリ秒単位レイヤの自動移動が開始されるまでの時間を ms で指定します。
省略すると 0 であるとみなされます。
なしaccelno-1 未満の数または 0(デフォルト) または 1 より大きい数加速度的な動きを行わせるかどうかを指定します。
0 を指定すると、最初から最後まで一定の割合で移動処理が進行し ます ( 各通過点を等間隔の時間で通過します )。省略すると 0 が指定され たと見なされます。
-1 未満の数を指定すると、最初は早く、徐々に遅くなります。
1 より大きい数を指定すると、最初は遅く、徐々に早くなります。
なしpositionnoミリ秒単位の整数またはポジション（0.0 〜 1.0）時間によるポジション指定の場合 time 属性の範囲内で指定します。
ポジションを直接指定する場合は 0.0 〜 1.0 の間で指定します。
なしxnoピクセル単位移動する位置を指定します。
なしynoピクセル単位移動する位置を指定します。
なしwidthnoピクセル単位移動先の画像幅を指定します。
なしheightnoピクセル単位移動先の画像高さを指定します。
なしixnoピクセル単位移動先の画像表示位置を指定します。
なしiynoピクセル単位移動先の画像表示位置を指定します。
なしopacityno0 〜 255移動先の不透明度を指定します。
なしangleno角度移動先の角度を指定します。
なしscalenoスケール移動先のスケールを指定します。
なしstartnotrue 又は falsetrue を指定すると移動を開始します。
false を指定すると何もしません。
デフォルトはfalseです。
レイヤを移動させます。
例:
[layer_move position=0.0 x=0 y=0]
[layer_move position=0.5 x=100 y=100]
[layer_move position=1.0 x=200 y=200 time=1000 start]
; レイヤ0を(100, 100)に移動して続けて(200, 200)に移動します。
