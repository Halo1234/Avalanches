move(No alias)

Layer operations
aliasattributeRequired?valueDescription
nonetimenomillisecondsSpecifies the time required for the layer to move a point from the point specified by path.
Therefore, the movement takes time equal to the number of intervals between points traversed times the value of this attribute.
However, if you specify a value other than 0 for the accel attribute, the time it takes to pass one point will change (the time required for the entire movement is the number of sections passed * the value of this attribute).
nonedelaynomillisecondsSpecifies the time in ms before automatic layer movement begins.
If omitted, it is assumed to be 0.
nonepathno"(x1,y1,opacity1) (x2,y2,opacity2) ..."Specify the movement position.
Specify the movement position by separating x (left edge position of layer), y (top edge position of layer), and opacity (layer opacity) with commas, spaces, or parentheses ( ).
opacity specifies the layer's display density. The density of the layer also changes continuously as you move from point to point.
noneaccelnoA number less than -1 or 0 (default) or greater than 1Specify whether to perform accelerated movement.
If you specify 0, the movement process will proceed at a constant rate from beginning to end (each passing point will be passed at equal intervals of time). If omitted, 0 is assumed.
A number less than -1 will start faster and gradually become slower.
A number greater than 1 will start slower and gradually speed up.
You can also specify a real number.
Move the layer.
Return to genre/tag list

layer_move(No alias)

Layer operations
aliasattributeRequired?valueDescription
noneclearnotrue or falseSpecifies whether to clear the travel path.
If you specify true, the movement path will be cleared.
If false is specified, nothing will be done.
Default is false.
nonetransformnotrue or falseSpecify whether to perform transformation (rotation scaling).
Specifying true will perform transformation.
false does nothing.
nonechildrennotrue or falseSpecifies whether to include child layers in the transformation.
If you specify true, children will also be included.
If false is specified, nothing will be done.
noneloopingnotrue or falseSpecify whether to loop.
Specifying true will loop.
If false is specified, nothing will be done.
nonetimenomillisecondsSpecifies the time required for the layer to move a point from the point specified by path.
Therefore, the movement takes time equal to the number of intervals between points traversed * the value of this attribute.
However, if you specify a value other than 0 for the accel attribute, the time it takes to pass one point will change (the time required for the entire movement is the number of sections passed * the value of this attribute).
nonedelaynomillisecondsSpecifies the time in ms before automatic layer movement begins.
If omitted, it is assumed to be 0.
noneaccelnoA number less than -1 or 0 (default) or greater than 1Specify whether to perform accelerated movement.
If you specify 0, the movement process will proceed at a constant rate from beginning to end (each passing point will be passed at equal intervals of time). If omitted, 0 is assumed.
A number less than -1 will start faster and gradually become slower.
A number greater than 1 will start slower and gradually speed up.
nonepositionnonteger or position in milliseconds (0.0 to 1.0)When specifying a position by time, specify within the range of the time attribute.
When specifying the position directly, specify between 0.0 and 1.0.
nonexnopixel unitSpecify the position to move.
noneynopixel unitSpecify the position to move.
nonewidthnopixel unitSpecify the image width to move to.
noneheightnopixel unitSpecify the image height to move to.
noneixnopixel unitSpecify the destination image display position.
noneiynopixel unitSpecify the destination image display position.
noneopacityno0 ～ 255Specify the opacity of the destination.
noneanglenoAngleSpecify the angle to move to.
nonescalenoScaleSpecify the scale to move to.
nonestartnotrue or falseIf you specify true, the movement will start.
If false is specified, nothing will be done.
Default is false.
Move the layer.
Example:
[layer_move position=0.0 x=0 y=0]
[layer_move position=0.5 x=100 y=100]
[layer_move position=1.0 x=200 y=200 time=1000 start]
; Move layer 0 to (100, 100) and then to (200, 200).
Return to genre/tag list
