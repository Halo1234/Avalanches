Default tag

A reference of tags available by default.
There are duplicate tags due to loading each module.
If there are duplicates, the tag of the module loaded later takes effect.

The syntax tag can be written as follows using the GConductor function.
@!tag /pattern=<TEST>_suffix &group=g1 element=name->value
