ModADVのタグ

　ModADV で使用可能なタグのリファレンスです。
このモジュールはModMessage/ModImage/ModSoundを必要とします。

ModADVにテキストを処理させる場合は[using_mod_adv]を呼び出します。
[using_mod_adv]を呼び出すとテキスト処理がch/ruby/r/p/er/cmタグが
ch_mod_adv/ruby_mod_adv/r_mod_adv/p_mod_adv/erase_message/clear_message_layerにリダイレクトされます。
テキストの処理をデフォルトに戻したい場合は[not_using_mod_adv]を呼び出します。

ModADVは特殊なテキスト解釈を行います。
メッセージを表示して改行が２個連続で現れると[p]を実行します。
キャラクタタグで立ち絵を表示させようとした場合、次のメッセージ表示直前にトランジションを自動で行います。
ModADVのテキスト処理は複雑なため詳細はソースを追ってください。
基本的には以下のような形式のシナリオにするとよいでしょう。

-----
[ハロ 通常 笑顔]シナリオテキストです。

[ハロ 通常 怒り]立ち絵が切り替わります。

[ハロ]ボイスがある場合は自動で再生されます。
-----

make_character(別名なし)

キャラクター操作
別名属性必須?値説明
なしnameyesキャラクター名作成するまたは設定を変更するキャラクター名を指定します。
ここで指定したキャラクター名がそのままタグ名になります。
なしmobnoモブかどうかキャラクターがモブかどうかを指定します。
true を指定するとモブキャラクターとなります。
モブキャラクターは定義されたシナリオファイル中でのみ有効なキャラクターとなります。
false を指定すると通常キャラクターとなり全シナリオファイル中で有効なキャラクターとなります。
デフォルトは false です。
なしauto_voiceno自動ボイス再生を行うかどうかキャラクターボイスを自動で再生するかどうかを指定します。
true を指定すると自動でボイス再生が行われます。
再生するボイスファイル名は voice_storage_format で指定したフォーマットが利用されます。
false を指定すると自動でボイス再生は行われません。
なしvoice_storage_formatno自動ボイス再生で再生するファイルフォーマット自動で再生するボイスファイル名のフォーマットを指定します。
このフォーマットには String#sprintf のフォーマットが利用できます。
ボイスのゼロから始まる通し番号が一つ引数として与えられます。
デフォルトは [キャラクター名]%03d です。
なしimageno立ち絵画像を持つかどうか立ち絵画像を持つかどうかを指定します。
立ち絵画像を持つ場合に true を指定します。
デフォルトは true です。
なしvoicenoボイスを持つかどうかボイスを持つかどうかを指定します。
ボイスを持つ場合に true を指定します。
デフォルトは true です。
なしmessagenoメッセージレイヤ番号キャラクタが使用するメッセージレイヤ番号を指定します。
なしvoice_buffernoボイスバッファ番号キャラクタが使用するボイスバッファ番号を指定します。
なしsub_directory_namenoサブディレクトリ名画像のサブディレクトリ名を指定します。
画像が/sub_directory/A/image001.pngにある場合sub_directoryを指定します。
この属性はpart_of_directory_pathと合わせて指定します。
なしpart_of_directory_pathno部分パス名画像の部分パス名を指定します。
画像が/sub_directory/A/image001.pngにある場合Aを指定します。
この属性はsub_directory_nameと合わせて指定します。
なしvoice_sub_directory_namenoサブディレクトリ名ボイスのサブディレクトリ名を指定します。
ボイスが/sub_directory/Scene001/ボイス001.oggにある場合sub_directoryを指定します。
Scene001の部分には実行中のシナリオファイル名（拡張子無し）が指定されます。
キャラクターの作成及び設定変更を行うことができます。
例:
[make_character name=ハロ voice_storage_format=ハロ_%03d]
; キャラクター「ハロ」のボイスファイル名を ハロ_連番 にする。

stage(make_stageで指定した名前)

背景表示
別名属性必須?値説明
なしnameno背景名表示する背景名を指定します。
なしpageno対象レイヤのページ対象レイヤのページ名。
fore を指定すると表ページのレイヤが対象になります。
back を指定すると裏ページのレイヤが対象になります。
なしchildrennotrue 又は falseトランジションに子も巻き込むかどうかを指定する。

省略すると他のパラメータから推測されます。

なしvaguenoあいまい領域値 (0〜)ユニバーサルトランジションに適用する あいまい 領域値を指定します。

なしstayno"stayfore" または "stayback" または "nostay"(デフォルト)スクロールトランシジョンにおいて、表ページの画像が その場にとどまったまま、裏ページの画像が外部から スクロールしてくるか ( "stayfore" )、
表ページの画像がスクロールして出ていく裏から、 裏ページの画像がその場にとどまったまま現れてくるか ( "stayback" )、
または表ページの画像と裏ページの画像が連結されて 一緒に移動しながら切り替わっていくか ( "nostay" )、を 指定します。
背景を表示します。
このタグを直接利用する事はないでしょう。
通常は背景名のタグを利用してください。
