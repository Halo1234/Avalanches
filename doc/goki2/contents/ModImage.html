レイヤ操作
別名属性必須?値説明
なしlayerno立ち絵レイヤの名前立ち絵レイヤの名前を指定します。
0 〜 N のいずれかになります。(N は cv_layers で指定した立ち絵レイヤの枚数 - 1)
省略するとbaseが指定されたものとみなします。
なしpageno対象レイヤのページ対象レイヤのページ名。
fore を指定すると表ページのレイヤが対象になります。
back を指定すると裏ページのレイヤが対象になります。
省略するとforeが指定されたものとみなします。
なしchildrennotrue 又は falseトランジションに子も巻き込むかどうかを指定する。
true を指定すると子も巻き込んでトランジションします。
false を指定すると単独でトランジションします。

"universal" を指定すると、 rule, vague, time 属 性に従ってユニバーサルトランジションが行われます。
"crossfade" を指定すると、time 属性に従って、単純な クロスフェードが行われます。
"scroll" を指定すると、from, stay を元にしてス クロールトランジションが行われます。
プラグインによって使用可能になるトランジションの タイプもあります。その場合は method 属性にはその トランジションの名前を指定し、また、トランジションによっては 別の属性を指定しなければならない場合があります。
省略すると他のパラメータから推測されます。
なしrulenoルール画像名ユニバーサルトランジションに用いるルール画像を指定します。
なしvaguenoあいまい領域値 (0〜)ユニバーサルトランジションに適用する あいまい 領域値を指定します。
なしfromno"left" または "top" または "right" または "bottom"スクロールトランジションにおいて、裏ページの画像がどの 方向から現れてくるかを指定します。"left" = 左、"top"= 上、 "right" = 右、"bottom" = 下
なしstayno"stayfore" または "stayback" または "nostay"(デフォルト)スクロールトランシジョンにおいて、表ページの画像が その場にとどまったまま、裏ページの画像が外部から スクロールしてくるか ( "stayfore" )、
表ページの画像がスクロールして出ていく裏から、 裏ページの画像がその場にとどまったまま現れてくるか ( "stayback" )、
または表ページの画像と裏ページの画像が連結されて 一緒に移動しながら切り替わっていくか ( "nostay" )、を 指定します。
指定したレイヤのトランジションを開始します。
