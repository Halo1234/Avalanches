Reference of tags available in ModBookmark.
This module requires ModSnapshot, but will work without it.

When [bookmark store_skipping=true] is executed, storage will not be performed while skipping.
To save, use the force_store label to force store before stopping with the [s] tag etc.
When the label name includes force_store, if the label can be saved, it will be forcibly stored.

bookmark(No alias)

Save data settings
aliasattributeRequired?valueDescription
noneformatnoStringSpecify the save data format.
If it is bmp8, it will be 8bit BMP.
If it is bmp24, it will be 24bit BMP.
Otherwise, it will be kdt.
nonewidthnoWidthSpecify the width of the thumbnail.
If it is auto, it will be calculated automatically.
Both height and height cannot be set to auto.
noneheightnoHeightSpecify the height of the thumbnail.
If it is auto, it will be calculated automatically.
Both height and width cannot be set to auto.
nonekeep_aspect_rationotrue or falseSpecifies whether to fix the aspect ratio of thumbnails.
If true, the aspect ratio will be fixed.
If false, the aspect ratio will not be fixed.
noneprefixnoStringSpecify the prefix name of the save data file.
nonesuffixnoStringSpecify the suffix name of the save data file.
nonemodenoModeYou can specify the save data mode.
Specifying c performs simple encryption.
If you specify z, compression will be applied.
Both z and c cannot be specified.
Specifying an offset after o will write from that point in the file.
nonestore_skippingnotrue or falseSpecify whether to store when skipping.
If true is specified, the store will not be executed when skipping.
If you specify false, the store will be executed when skipping.
Default is false.
If force_store is included in the label name, it will be forced to store even if this attribute is true.
nonestore_and_savenotrue or falseSpecify whether to store or save.
If you specify true, store and save will be performed.
If you specify false, neither store nor save will be performed.
Default is true.
nonequicknoStringSpecify the quick save data file name.
Configure settings related to save data.
