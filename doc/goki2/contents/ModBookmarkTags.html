ModBookmarkのタグ

　ModBookmark で使用可能なタグのリファレンスです。
このモジュールはModSnapshotを必要としますが、無くても動作はします。

[bookmark store_skipping=true]を指定した時、スキップ中はストアを行いません。
セーブするためには[s]タグなどで停止する前にforce_storeラベルを使用して強制的にストアしてください。
ラベル名にforce_storeが含まれる時、そのラベルがセーブ可能な場合、強制的にストアされます。

false ならばアスペクト比は固定されません。

なしsuffixnoセーブデータファイルのサフィックスセーブデータファイルのサフィックス名を指定します。
なしmodenoセーブデータのモードセーブデータのモードを指定できます。
c を指定すると簡単な暗号化が施されます。
z を指定すると圧縮がかかります。
z と c を両方指定する事はできません。
o の後にオフセットを指定するとファイルのその位置からの書き込みになります。
