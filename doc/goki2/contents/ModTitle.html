title_option(別名なし)

効果音操作
別名属性必須?値説明
なしstoragenoストレージ名システム画面の背景に使用する画像を指定します。
なしbufferno効果音バッファ番号効果音を再生するバッファ番号を指定します。
なしclick_soundnoストレージ名クリック時に再生する音声ファイル名を指定します。
なしenter_soundnoストレージ名マウス侵入時に再生する音声ファイル名を指定します。
なしleave_soundnoストレージ名マウスが出て行った時に再生する音声ファイル名を指定します。
なしshow_hide_typenovista or fade表示タイプを指定します。
タイトル画面の設定を変更します。

title_screen(別名なし)

効果音操作
別名属性必須?値説明
なし<id>_button_storagenoストレージ名ボタンに使用する画像を指定します。
<id>には以下の文字列が使用できます。
start : ゲーム開始
load : ロード画面
system : システム設定画面
cgmemory : 未実装
exit : ゲーム終了
なし<id>_leftno位置ボタンの位置を指定します。
なし<id>_topno位置ボタンの位置を指定します。
なし<id>_widthnoサイズボタンのサイズを指定します。
なし<id>_heightnoサイズボタンのサイズを指定します。
なし<id>_captionno文字列ボタンのキャプションを指定します。
なし<id>_caption_colorno0xRRGGBBボタンのキャプションカラーを指定します。
なし<id>_colorno0xRRGGBBボタンの背景色を指定します。
なし<id>_opacityno0〜255ボタンの不透明度を指定します。
なし<id>_storagenoストレージ名ボタンを選択した時にジャンプするシナリオストレージ名を指定します。
この項目はidがstartとexitの時のみ有効です。
なし<id>_targetnoストレージ名ボタンを選択した時にジャンプするラベル名を指定します。
この項目はidがstartとexitの時のみ有効です。
タイトル画面の項目を設定します。
