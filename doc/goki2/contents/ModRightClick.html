right_click(rclick)

Right click settings
aliasattributeRequired?valueDescription
noneenablednotrue or falseIf set to true, right-clicking will hide the message layer, call the specified routine, or jump to the specified location.
Specifying false prevents these actions.
nonejumpnotrue or falseIf set to true, right-clicking will jump to the location specified by the storage and target attributes. Specifying false for both the call attribute and this attribute results in normal ``disappear message'' behavior.
This attribute and the call attribute cannot be true at the same time.
nonecallnotrue or falseIf set to true, right-clicking will call the subroutine specified by the storage and target attributes. If you specify false for both the jump attribute and this attribute, the normal "delete message" behavior will occur.
This attribute and jump attribute cannot be set to true at the same time.
nonestoragenoStorage nameIf the call or jump attribute is set to true, specify the file to call/jump to.
If this attribute is set to storage="", the label in the file at the time of right-clicking will be searched.
nonetargetnoLabel nameIf the call or jump attribute is set to true, specify the label of the call destination/jump destination.
If this attribute is set to target="", execution will start from the beginning of the file specified by the storage attribute.
Set the behavior when right-clicking.
If any attribute is omitted, the settings will not be changed.
